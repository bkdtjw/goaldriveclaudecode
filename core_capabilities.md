# 核心能力说明

## 1. 文本理解能力
- **语义理解**：准确理解用户输入的文本含义，包括上下文、隐含意图
- **多语言理解**：支持多种语言的理解和跨语言处理
- **结构化解析**：从非结构化文本中提取关键信息、实体和关系

## 2. 文本生成能力
- **内容创作**：撰写文章、报告、邮件、创意写作等各类文本
- **代码生成**：根据需求生成多种编程语言的代码
- **翻译转换**：在不同语言之间进行准确翻译
- **摘要总结**：将长文本压缩为简洁的要点摘要

## 3. 推理分析能力
- **逻辑推理**：基于已知信息进行逻辑推断和判断
- **问题求解**：分析复杂问题并提供解决方案
- **数据分析**：解读数据、发现模式、提供洞察
- **批判性思维**：评估信息质量，识别偏见和逻辑谬误

## 4. 编程辅助能力
- **代码审查**：检查代码质量、发现潜在bug
- **调试协助**：帮助定位问题、提供修复建议
- **技术解释**：解释算法、框架和编程概念
- **项目架构**：提供软件设计建议和最佳实践

## 5. 对话交互能力
- **上下文保持**：在多轮对话中维持连贯的上下文
- **个性化响应**：根据用户需求调整回答风格
- **澄清追问**：在信息不足时主动询问以获取必要信息
